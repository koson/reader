BBeB Binder Release Notes

Introduction

BBeB Binder is a program for converting text and HTML documents into unencrypted Broadband eBook files - or BBeB files. These files are read by the Sony Reader (PRS-500).

Note: BBeB Binder is currently a work-in-progress. There is still plenty of HTML formatting that is not properly converted to the BBeB format.

Installation

To install BBeB Binder run setup.exe which is beside BBeBinderSetup.msi. BBeB Binder is a Microsoft .Net 2.0 application, and setup.exe will download and install the .Net 2.0 runtime if it detects that it is not already installed.

Note: If you run BBeB Binder the first time and a "filenotfound" exception is displayed then you need to run the vs_piaredist.exe program which accompanies this setup program. This will hopefully be integrated into a future release of the BBeB Binder installer.

Using BBeB Binder

Creating an eBook with BBeB Binder is simple. Just do the following:

  1. Open the text or HTML page using the File/Open menu item.
  2. Make any changes to the eBook properties if necessary.
  3. Make any changes to the HTML document (if necessary).
  4. Select the File/Convert menu item and specify the destination file.

This will produce a BBeB file (*.lrf) in the directory of your choosing. If you have the Sony CONNECT reader installed then double clicking on your newly created file should open and display that new eBook. You can then copy this new eBook to your reader as you would any other book.

Releases

0.2.0

  • Added support for creating images in BBeB files.
  • Added support for reading existing BBeB files.
  • Can now open HTML pages from the Internet instead of just local HTML files.
  • Now create a table of contents automatically based on the HTML heading elements: i.e. H1, H2, etc.
  • Create a very simple book cover.
  • Added a character substitution file that can be used to specify characters to use when none exists in the Sony Reader.
  • H1 and H2 elements now cause a new page to be created.

0.1.5

  • First releases. Supported simple text creation
